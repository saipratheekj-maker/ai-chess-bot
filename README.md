# ♟ AI Chess Bot using Minimax, Alpha-Beta Pruning & PyTorch

An advanced AI-powered Chess Bot website built using **Python, Flask, HTML, CSS, JavaScript, and PyTorch**.
The project implements the **Minimax Algorithm with Alpha-Beta Pruning** to generate intelligent chess moves and uses a **PyTorch neural network** for enhanced board evaluation.

---

# 🚀 Features

* Interactive Click-to-Move Chess Board
* AI-based Chess Opponent
* Minimax Algorithm Implementation
* Alpha-Beta Pruning Optimization
* PyTorch Neural Network Integration
* Multiple Difficulty Levels:

  * Beginner
  * Easy
  * Medium
  * Hard
  * Expert
* Responsive Modern UI
* Flask Backend Integration
* Real-time AI Move Generation
* Chess Position Evaluation
* Restart Game Functionality

# 🧠 AI Concepts Used

## 1. Minimax Algorithm

The AI predicts future moves by exploring possible game states and choosing the optimal move.

## 2. Alpha-Beta Pruning

Optimizes the Minimax Algorithm by eliminating unnecessary branches in the game tree, improving performance.

## 3. PyTorch Neural Network

Used for evaluating board positions and improving strategic move analysis.

# 🎮 How to Play

1. Open the website in your browser.
2. Select a difficulty level.
3. Click a white chess piece.
4. Click the destination square.
5. The AI automatically calculates and plays its move.
